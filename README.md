# 📈 Signalist

Signalist is a modern stock tracking and portfolio management web application built with Next.js.
It enables users to monitor stock prices, analyze historical trends, and manage their investment portfolio through a fast and responsive interface.

---

## 🚀 Features

### 📊 Stock Tracking
- Search and track listed stocks
- View real-time or near real-time price data
- Monitor daily change and percentage movement

### 📈 Trend Analysis
- Interactive historical price charts
- Timeframe filtering (1D, 1W, 1M, 1Y)
- Basic performance insights

### 💼 Portfolio Management
- Add and remove holdings
- Track invested capital
- Calculate unrealized profit/loss
- View total portfolio value

### ⚡ Performance & UX
- Optimized with Next.js App Router
- Server-side data fetching where applicable
- Fully responsive UI
- Clean and minimal design system

---

## 🛠 Tech Stack

- **Next.js** – Full-stack React framework
- **React** – Component-based UI
- **TypeScript** – Type safety and scalability
- **Tailwind CSS** – Utility-first styling
- **REST API Integration** – Market data provider

---

## 🏗 Architecture Overview

- App Router (Next.js)
- Server Components for data fetching
- Client Components for interactivity
- API abstraction layer for stock data
- Modular component structure
